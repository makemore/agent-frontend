A standalone chat widget that can be embedded in any website with just a few lines of code.

Quick Start

Add the widget to your website by including the CSS and JavaScript files, then initialize with your configuration:

<!-- Include the widget files -->
<link rel="stylesheet" href="chat-widget.css">
<script src="chat-widget.js"></script>

<!-- Initialize the widget -->
<script>
  ChatWidget.init({
    backendUrl: 'https://your-api.com',
    agentKey: 'your-agent',
    title: 'Support Chat',
    primaryColor: '#0066cc',
  });
</script>
