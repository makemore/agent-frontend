Chat Widget - Text-to-Speech Test

This page tests the ElevenLabs TTS integration.

Setup Instructions:

  1. Get your ElevenLabs API key from elevenlabs.io
  2. Get voice IDs from the Voice Library
  3. Update the config below with your API key and voice IDs
  4. Click the speaker icon in the chat header to toggle TTS
